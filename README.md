📊 Dashboard de Vendas | Power BI
📌 Sobre o Projeto

Este projeto tem como objetivo desenvolver um dashboard interativo no Power BI para análise de dados de vendas, permitindo a visualização de métricas essenciais e geração de insights para apoio à tomada de decisão.

🎯 Objetivos do Projeto

Monitorar o faturamento total
Identificar os produtos mais vendidos
Analisar o desempenho por categoria
Avaliar a distribuição de vendas por cidade
Criar uma visualização clara e interativa dos dados

📁 Dataset

O conjunto de dados utilizado é fictício, contendo informações de vendas com as seguintes colunas:

Data
Produto
Categoria
Preço
Quantidade
Cidade

🛠️ Ferramentas Utilizadas
Power BI
Excel

📊 Modelagem de Dados
Foi criada uma métrica essencial para análise:
Faturamento = Preço × Quantidade
Essa métrica foi utilizada como base para todos os indicadores do dashboard.

📈 Visualizações Desenvolvidas
O dashboard foi estruturado com os seguintes elementos:

📌 Cartão (KPI): Faturamento Total

📊 Gráfico de Barras: Produtos mais vendidos

🥧 Gráfico de Pizza: Participação por categoria

🗺️ Tabela/Mapa: Vendas por cidade

📅 Filtro (Slicer): Data

📊 Principais Insights

A categoria Eletrônicos apresentou maior contribuição no faturamento total
Determinadas cidades concentram maior volume de vendas, indicando potencial de mercado
Produtos com maior valor agregado geram maior impacto na receita

🚀 Como Visualizar o Projeto

Baixe o arquivo .pbix disponível neste repositório
Abra utilizando o Power BI Desktop
Interaja com os filtros para explorar os dados
